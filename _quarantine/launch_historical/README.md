# Historical Launch Jobs

This directory contains Slurm launchers moved out of `launch/` during cleanup step 2.

These jobs are historical experiments or old protocol variants, including:

- `anchor_max`
- `ampphase`
- `modefocus`
- `highalpha`
- `singlecase`
- `first_order_real`
- `edge_repair`
- old `riccati_*` jobs outside the active Stage 0 / Stage 1 / Stage 1bis protocol
- old `M00`, `M03`, `M07` launchers
- old `2d` launchers not used in the current protocol
- broken or obsolete launchers such as `ci_supervision_budget`
- old generic launchers `jz_submit.slurm` and `jz_submit_cpu.slurm`

No file was deleted. Files were moved with `git mv`.
